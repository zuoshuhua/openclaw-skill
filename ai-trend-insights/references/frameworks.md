# 趋势研判分析框架

## 1. Gartner 技术成熟度曲线定位

用于判断某项 AI 技术当前所处阶段：

```
创新触发期 → 期望膨胀期 → 泡沫破裂期 → 稳步爬升期 → 生产成熟期
```

| 阶段 | 判断信号 | 当前典型技术（2025） |
|------|----------|---------------------|
| 创新触发期 | 学术论文多、产品少 | 量子+AI融合、脑机接口AI |
| 期望膨胀期 | 媒体热炒、融资激增 | AI Agent、具身智能 |
| 泡沫破裂期 | 落地不及预期、裁员 | 部分AI硬件产品 |
| 稳步爬升期 | 垂直场景成功、商业化 | 代码生成、AI客服 |
| 生产成熟期 | 大规模部署、标准化 | 图像识别、语音识别 |

---

## 2. PEST 政策-经济-社会-技术框架

用于结构化分析宏观趋势驱动因素：

- **P（政策）**：AI监管收紧/放开、政府补贴、国家战略
- **E（经济）**：AI投资规模、降本增效ROI、就业影响
- **S（社会）**：公众接受度、伦理争议、人才供需
- **T（技术）**：算力成本曲线、模型能力边界、开源生态

---

## 3. 竞争格局分析维度（AI领域专用）

| 维度 | 关键问题 |
|------|----------|
| 模型能力 | Benchmark排名变化？能力边界突破？ |
| 成本曲线 | 推理成本同比下降多少？训练成本趋势？ |
| 生态锁定 | API调用量、开发者数量、插件/工具数 |
| 数据飞轮 | 私有数据优势、RLHF反馈规模 |
| 监管适应性 | 合规能力、隐私保护架构 |

## 4. 趋势信号强弱判断

| 信号类型 | 强度 | 描述 |
|----------|------|------|
| 多家头部公司同向行动 | ★★★★★ | 行业共识形成 |
| 重大资本流入（>$1B） | ★★★★☆ | 市场验证 |
| 政策明确支持/限制 | ★★★★☆ | 外部约束确定 |
| 单一公司技术突破 | ★★★☆☆ | 需观察跟进 |
| 媒体热度激增 | ★★☆☆☆ | 可能是噪音 |
| 个人/KOL观点 | ★☆☆☆☆ | 仅供参考 |
